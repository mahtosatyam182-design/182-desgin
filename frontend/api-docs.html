API Overview

Welcome to the ShopMax E-Commerce API documentation. This API provides access to products, categories, users, orders, and reviews for building e-commerce applications.

Base URL

http://localhost:3000/api

Response Format

All responses are returned in JSON format with the following structure:

{ "success": true, "data": { ... }, "count": 10, "page": 1, "totalPages": 5 }

Error Format

{ "success": false, "error": "Error message", "message": "Detailed error information" }

Authentication

Most API endpoints require authentication using JWT (JSON Web Tokens). Include the token in the Authorization header.

Authentication Header

Authorization: Bearer

Getting a Token

Obtain a JWT token by logging in with your credentials:

POST /users/login

Authenticate user and receive JWT token

Request Body

{ "email": "user@example.com", "password": "password123" }

Response

{ "success": true, "data": { "id": 1, "name": "John Doe", "email": "john@example.com", "token": "eyJhbGciOiJIUzI1NiIsInR...", "role": "user" } }

Products API

Manage products in the store catalog.

GET /products

Get all products with pagination and filters

Query Parameters

pagePage number (default: 1)
limitItems per page (default: 10)
searchSearch term for name/description
categoryFilter by category ID
minPriceMinimum price filter
maxPriceMaximum price filter
featuredShow featured products only
sortByField to sort by (name, price, id)
sortOrderasc or desc
GET /products/:id

Get product by ID

GET /products/featured

Get featured products

POST /products

Create new product (Admin only)

PUT /products/:id

Update product (Admin only)

DELETE /products/:id

Delete product (Admin only)

Categories API

Manage product categories.

GET /categories

Get all categories

GET /categories/:id

Get category by ID

POST /categories

Create category (Admin only)

PUT /categories/:id

Update category (Admin only)

DELETE /categories/:id

Delete category (Admin only)

Users API

User authentication and profile management.

POST /users/register

Register new user

POST /users/login

Login user

GET /users/profile

Get current user profile (Auth required)

PUT /users/profile

Update user profile (Auth required)

GET /users/orders

Get user orders (Auth required)

Orders API

Order management and tracking.

GET /orders

Get user orders (Auth required)

GET /orders/:id

Get order by ID (Auth required)

POST /orders

Create new order (Auth required)

PUT /orders/:id/status

Update order status (Admin only)

POST /orders/:id/cancel

Cancel order (Auth required)

Reviews API

Product review management.

GET /reviews/product/:productId

Get product reviews

POST /reviews

Add review (Auth required)

DELETE /reviews/:id

Delete review (Auth required)
