API Overview

Welcome to the ShopMax E-Commerce API. This API provides access to products, categories, users, orders, and reviews for building e-commerce applications.

Base URL

/api

Response Format

{ "success": true, "data": { ... }, "count": 10 }

Authentication

Most API endpoints require authentication using JWT tokens. Include the token in the Authorization header.

Authentication Header

Authorization: Bearer
POST /users/login

Authenticate user and receive JWT token

Request Body

{ "email": "user@example.com", "password": "password123" }

Products API

GET /products

Get all products with pagination & filters

GET /products/:id

Get product by ID

GET /products/featured

Get featured products

Categories API

GET /categories

Get all categories

GET /categories/:id

Get category by ID

Users API

POST /users/register

Register new user

POST /users/login

Login user

GET /users/profile

Get user profile (auth required)

GET /users/orders

Get user orders (auth required)

Orders API

GET /orders

Get user orders (auth required)

POST /orders

Create new order (auth required)

POST /orders/:id/cancel

Cancel order (auth required)
